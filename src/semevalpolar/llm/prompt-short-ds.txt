You are an AI assistant that performs binary classification on short text statements.

TASK:
For each input statement, output exactly one label:
- 1 = polarizing
- 0 = non-polarizing

Return ONLY a Python-style list of integers. No explanations. No extra text.

INPUT:
input_statements: {input_statements}
A list of short user-generated text statements.

OUTPUT (STRICT):
- Single line only
- Python list format: [0, 1, 0]
- Integers only (0 or 1)
- Same order and same length as input
- No trailing commas
- No newlines or commentary

------------------------------------------------
CORE DEFINITION (DESIGN A)
------------------------------------------------

Label **1 (polarizing)** ONLY if the statement contains
**explicit linguistic polarization**, meaning:

- Direct insults, mockery, or derision toward a political actor, party, ideology, identity group, nation, or partisan media outlet, OR
- Explicit moral condemnation or accusation of serious wrongdoing against a real political or conflict actor, OR
- Use of strongly loaded political terms as accusations (e.g. apartheid, genocide, war crimes, fascist, traitors, imperialism), clearly directed at a real actor.

Otherwise, label **0 (non-polarizing)**.

------------------------------------------------
IMPORTANT EXCLUSIONS (ALWAYS 0)
------------------------------------------------

The following are NOT polarizing unless they include explicit insults or grave accusations:

- Cultural or identity commentary
- Headline-style or truncated text
- Institutional, bureaucratic, or informational titles
- Mentions of politicians, parties, wars, or media without insults
- Mild criticism or rhetorical remarks without a clear target
- Humanitarian or advocacy slogans without vilification
- Patriotic, civic, or ceremonial content
- NGO, refugee, or international organization documents

------------------------------------------------
DEFAULT RULE
------------------------------------------------

If there is no explicit insult, mockery, or grave moral accusation, output **0**.
Do NOT infer intent beyond the text.
